Process Analyses

1.

Describe


1. How would you describe and explain P2?
It directly branded the E-commerce website[TROS], which sells simple, stylish clothes and bags, and designed the entire site layout into detailed pages. After creating a style guide divided by design components, HTML/CSS/Javascript built a complete and unified website.


Analyze


2. What might be some 'problems' encountered?
I worked on the web version based on what is shown on the screen based on HTML and CSS. Because we worked on the framework without configuring wireframing in advance, we experienced structural confusion as the content increased.


Evaluate



3. What did you learn?
When I organized the website, I realized it was necessary to frame the whole thing and develop it individually. Also, I learned how to change the configuration as I intended when I was confused with an unwanted design.



4. How would you explain them to our audiences?
This is the first project that I branded and built using HTML/CSS/Javascript. Through this project, I understood how to build a website and how to build a responsive website.



2.


Describe


1. How would you describe and explain UI/UX project, root?
Root application designed for seniors over 75 with families abroad who want to connect with their families.


Analyze


2. What might be some 'problems' encountered?
As our potential user group includes seniors, we had to carefully organize all the designs so that they were more considerate than we see in the modern world. That's why when I inadvertently found a page consisting of familiar designs and font sizes, I repeatedly modified it again.


Evaluate


3. What did you learn?

For the uniformity of the design, it would be helpful to have a detailed style guide at the beginning of the UI/UX project.


4. How would you explain them to our audiences?
It was a project I could create about a design that uses Figma and proto-pie and improves the user experience.

Project 1 - ROOT

ROOT is a family social communicating application that allows users to engage, connect, and share updates or news with their families abroad. Designed for easier accessibility for seniors. ROOT was a mobile application project from one of my classes, IAT 334: Interface Design, at Simon Fraser University.

Project 2 - SNAP

Our group was tasked to redesign the interface for the dining feature in the SFU Snap app that allows its users to find information about dining on SFU campus. This was a mobile application project from one of my classes, CMPT 363: User Interface Design, at Simon Fraser University. During this project, I was in charge of user research, developing new features, and re-designing and prototyping via Figma.
